# ARGOS-Labs POT SDK2 on Windows, Linux, Mac

## English Version

Welcome to the Python-to-Operation Toolset (POT) SDK2 for ARGOS Low-code platform. This document will show you how to develop and release a plugin from your Python solution. 

The most big difference from previous version are:

* Any python version can be supported from 3.8 (default version is 3.11)
* Not only Windows 10/11 but also support Linux and Mac system


[Start Here](POT_SDK2_en.md)
